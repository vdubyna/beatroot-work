# BEETROOT LLM Wiki

Це локальна wiki для LLM і людини, яка працює з навчальними embedded-проєктами
в цьому репозиторії.

## Як читати

1. `wiki/llm-context.md` - короткий snapshot для агента перед будь-якою роботою.
2. `wiki/hardware/yd-esp32-23.md` - основна плата, GPIO, USB, живлення, ризики.
3. `wiki/hardware/parts.md` - доступні або вже використані деталі.
4. `wiki/tools/` - локальні інструменти розробки та вимірювання.
5. `wiki/projects/index.md` - карта прикладів, домашок і використаних пінів.
6. `wiki/workflows/` - як створювати нові Rust-проєкти і як перевіряти залізо.
7. `reference/` - локальні даташити, схеми, pinout-зображення.

## Правила підтримки wiki

- Якщо змінюється плата, пін, інструмент, порт або підключення - оновити wiki в
  тому ж коміті/зміні.
- Факти маркувати за надійністю: `confirmed on board`, `photo`, `datasheet`,
  `community source`, `assumption`.
- Для ESP32-S3 не копіювати поради з класичного ESP32 без перевірки.
- Для зовнішніх джерел не використовувати `.ru` домени або російськомовні
  матеріали, якщо користувач прямо цього не попросив.
- Довгі PDF не вставляти в промпти. Давати локальний шлях і короткий висновок.

## Історичний журнал

`hardware-log.md` залишено як журнал робіт і перевірок. Нові короткі знання для
LLM треба заносити в `wiki/`, а детальні експериментальні записи можна лишати в
журналі.
